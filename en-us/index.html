Thain

Thain is a distributed flow schedule platform.

Quick StartView on Github

A distributed and easy-to-expand visual DAG workflow scheduling system

Dedicated to solving the complex dependencies in data processing, making the scheduling system out of the box for data processing. Its main objectives are as follows:

Feature List

  • Plugins

    Customizing a plugin requires only a simple java class.

  • High availability

    You don't need to configure a framework such as ZooKeeper, you can HA yourself, don't worry about the machine suddenly hangs.

  • Easy to scale

    Scale out, you only need to deploy a new server.

  • Easy to use

    Going to the simple, only retaining the most core functions.

  • HTTP call

    Comes with the SDK and can be used with other systems.

  • SLA

    The duration of the task is fully guaranteed.

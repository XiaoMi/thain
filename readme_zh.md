## 产品简介

Thain是小米自研的新一代分布式任务调度平台，提供定时、任务编排、分布式跑等功能。
Thain提供了任务调度与执行的一整套解决方案，在小米集团内部使用并久经考验，具有易学习、易上手、开发高效稳定的特点。
有完善的后台管理界面，支持任务的依赖，重跑，回溯，任务执行情况查看。
